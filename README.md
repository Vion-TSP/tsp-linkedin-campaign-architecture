## What's Inside

Interactive HTML page documenting The Sales People's account-based LinkedIn campaign architecture:

- **4 campaigns** targeting 3,099 companies (merged OLD + NEW lists)
- **Campaign segmentation** by company need (Founder-Led, Add Outbound, Optimise Engine, GTM Efficiency)
- **34 content assets** mapped to campaigns with segment targeting details
- **8 TLA post drafts** (full copy, ready for Sully's LinkedIn)
- **20 organic post drafts** (full copy, 8-week posting schedule)
- **3 video scripts** with detailed production briefs
- **Case study & testimonial strategy** with client assessment
- All content backed by C&S transcript data (1,401 conversations, 10,024 dial attempts)
